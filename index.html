Q3. Write code to implement timer clock using JS -- display HH:MM:SS -- the time should keep updating every second.

=>

        Code:

        let clockDisplay = document.getElementById("clockDisplay");
        clockDisplay.addEventListener('click', ()=>{
            clock()
        })
        function clock(){
            let time = new Date().toLocaleTimeString();
            clockDisplay.innerHTML = time
            
            setInterval(() => {
                    clock();
                }, 1000);
        }

Output:

Display Clock
